Description

Enumerate is an app designed to assist with the memorisation of word translations for foreign languages. It accomplishes this by providing flashcards and translate tasks based on a manually written or parsed word list.

Details

Skills: Swift
Scope: Personal
Date: August 2023

Features

  • Excessive Repetition: Splits a provided wordlist into multiple 'milestones'. Each 'milestone' contains 10 new words which are learned at the beginning through flashcards. The rest of the cards in the 'milestone' are translate tasks for every previously encountered word. When using a word list based on how common a specific word is, this results in more common words being repeated often - hopefully resulting in better memory achievements.
  • List Parsing: Enumerate has an inbuilt parser for wordlists. It was designed for lists found on certain language learning websites, but can also work with manually created ones. Through the use of this parser, the time needed to configure the desired wordpairs becomes trivial - giving more time for actual memorisation.
  • Text to Speech: Enumerate uses the iPhone's accessibility settings to create audio feedback when going through the cards (any card in a foreign language is read out by the TTS voice configured).
