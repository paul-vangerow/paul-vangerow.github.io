Monte Carlo Localised Robot

  1. Home
  2. Monte Carlo Localised Robot

Description

A six person group project with the objetive of creating a Lego Mindstorms, mobile robot capable of using Monte Carlo Localisation to determine its own position and navigate between a set of points within a Known Maze.

Details

Skills: Python
Scope: University
Date: March 2023

Responsibilities

My main responsibilities for this project were in creating an extensible platform to control the robots locomotion.

  • Used OOP Practices to create a configurable library for robot locomotion, implementing functions for Driving a set distance and turning a specified angle.
  • Created helper functions for drawing to the interface provided to use for the visualisation of the points generated by the monte carlo localisation algorithm.
